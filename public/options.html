Voice Browser Helping Guide

Note
  • This options page should always be opened to use this extension. To close this extension, please close this page.
  • Please open only one browser window. Close all other windows if open. This extension supports only one window with multiple tabs.

Root

Root is the entry module. From this module you can go to all other by saying their respectuive module names.

Commands Desired Action
Scroll Down Scrolls down the webpage
Scroll Up Scrolls up the webpage
Scroll Right Scroll the webpage right
Scroll Left Scroll the webpage left
Top Goes to the extreme top of the webpage
Bottom Goes to the extreme bottom of the webpage
Zoom In Increase webpage zoom by 10%
Zoom Out Decrease webpage zoom by 10%
Go Forward Goes forward if exist
Go Back Goes back if exist
Reload Reloads the webpage
Previous Tab Move to previous tab
Next Tab Move to next tab
New Tab Opens a new tab
Close Tab Closes the current tab

Auto Scroll

When you are in the root module, say Auto Scroll command to use this module

Commands Desired Action
Down Automatically scrolls down the webpage
Up Automatically scrolls up the webpage
Right Automatically scroll the webpage right
Left Automatically scroll the webpage left
Stop Stop scrolling
Faster Increase speed of scrolling
Slower Decrease speed of scrolling
Exit Exit from this module and go to root module
Just try by telling Auto Scroll

Scroll Area

When you want to scroll a sidebar or a menubar or any scrollable inner webpage content, you can use this module to scroll. When you are in the root module, say Scroll Area command to use this module. All scrollable area will be highlighted and an ID will be assigned, select which scroll area you want to scroll by telling the respective ID

Commands Desired Action
Down Scrolls down the selected area
Up scrolls up the selected area
Right Scrolls the selected area right
Left Scrolls the selected area left
Auto Down Automatically scrolls down the selected area
Auto Up Automatically scrolls up the selected area
Auto Right Automatically scroll the selected area right
Auto Left Automatically scroll the selected area left
Stop Stop automatic scrolling
Faster Increase speed of automatic scrolling
Slower Decrease speed of automatic scrolling
Exit Exit from this module and go to root module
Scroll me! Using Scroll Area option. I am partially visible. To see my full content you need to scroll this area. ForHorizontalScrollImadethistextwithoutspace
Just try by telling Scroll Area

Open URL

When you want to open a particular URL or website, you can use this module. When you are in the root module, just say Open command to use this module. The module shall request you to say the URL or website name or address.

Just try by telling Open

Search

When you want to search something, you can use this module. When you are in the root module, just say Search command to use this module. The module shall request you to say the search query.

Just try by telling Search

Click

You can use this module when you want to click a link or a button, enter text, select checkbox, control media, slide a range option, select in dropdown, select a color, select a datetime, month, date, time, week. When you are in the root module, just say Click command to use this module. All clickable elements will be highlighted with an ID, select which element you want to click by telling its respective ID

Scenarios Description
For link or a button Clicks it
For writing based input elements Makes the cursor to point to that element and gives Dictate Text module for entering text
For range input element Makes the cursor to point to that element and gives Range module for selecting range
For color input element Gives Color Picker module for selecting color
For DateTime input element Gives DateTime Picker module for selecting datetime
For Date input element Gives Date Picker module for selecting date
For Month input element Gives Month Picker module for selecting month
For Week input element Gives Week Picker module for selecting week
For Time input element Gives Time Picker module for selecting time
For Dropdown based element Gives Select module for selecting options
For Video or Audio element Gives Media Control module for controlling video or audio
New webpage
Just try by telling Click

Dictate Text

When you want to write or enter something, you can use this module. When you are in the click module, and you have selected a text based writable element, then you will get this module.

Command Desired Action
Keyboard Opens Keyboard for entering text
Remove Word Removes last word
Remove line Remove current line
Clean Up Delete all the entered text
Paste Paste the copied content
Submit If the element is inside a form element, submits the enclosed form.

Just try by telling click and select the above input element

Media Control

When you want to control a video or audio like play, pause or rewind, you can use this module. When you are in the click module, and you have selected a video or audio element, then you will get this module.

Command Desired Action
Play Plays a video or audio
Pause Pauses a video or audio
Stop Continues from the beginning
Mute Mutes Volume
Unmute Unmutes volume
Volume Up Increases volume by 10%
Volume Down Decreases volume by 10%
Rewind Rewinds 10 seconds
Forward Forwards 10 seconds

Just try by telling Click and select the above media element

Select

When you want to click a dropdown and select anyone option, you can use this module. When you are in the click module, and you have selected a select element or dropdown, then you will get this module.

Scenario Description
Number Tell the ID of the option to select it
Scroll Up Scroll the area up if area overflowed
Scroll Down Scroll the area down if area overflowed

Just try by telling Click and select the above dropdown
